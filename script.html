This is IRB. It is a command-line Read-Execute-Print loop for ruby, which basically means it reads a line of ruby, runs it, and then spits the result back out.

Ruby is a programming language very similar to python.

It does simple math

2 + 3
49 / 7 + 2

Call a function by typing its name. Parentheses are not required.

puts 'Hello, world!'
puts( 'Hello, world!' )

You can do some interesting one-liners:

5.times { puts 'Hey, listen!'; sleep 1 }

Demo - Sinatra

clear and require 'sinatra'

get '/hello' do
    'Hello, world!'
end 

Sinatra::Application.run!

Open a browser and navigate to http://localhost:4567/hello. ^C when finished.

get '/greet/:name' do |name|
    'Hello, ' + name + '!'
end

Sinatra::Application.run!

Open a browser and navigate to http://localhost:4567/greet/RC. ^C when finished.

Note that it is really easy to define routes - I am just showing you get because it is easy, but you can also play with request and response properties, use post, put, delete -- it is really easy to build a RESTful API with this.
